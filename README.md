# Employee Workforce Monitoring Dashboard (Excel)
## 📌 Project Type
Personal Project (Portfolio)
## 📊 Overview
This project presents as interactive Employee & Workforce Monitoring Dashboard built using Microsoft Excel. It focused on transforming raw employee data into structured and easy-to-analyze information to generate useful insights related to workforce distribution, performance (KPI), and compensation. The process includes data cleaning, simple transformations, and preparing the dataset to ensure it is ready for analysis and reporting.
## 🎯 Key Objectives
- Analyze workforce distribution across locations, divisions, and job roles
- Evaluate employee performance using KPI metrics
- Assess salary and bonus allocation across positions
- Identify workforce trends over time
- Explore employee demographics (age and gender)
